# 🌅 Recomeçar

Recomeçar é uma experiência de leitura guiada para pessoas que estão vivendo o pós-término ou momentos de transição emocional.

Aqui não há promessas vazias ou frases prontas.  
Há reflexão, consciência e um caminho leve de retorno para si mesmo.

## 💭 Propósito

Ajudar você a enxergar com mais clareza os ciclos emocionais que te prendem e abrir espaço para um novo começo — mais consciente, mais leve e mais seu.

## 🧠 O que você vai encontrar

- Reflexões sobre vínculos emocionais
- Compreensão de padrões afetivos
- Clareza sobre apego e saudade
- Caminhos para reconstrução emocional

## 🌱 Para quem é

Para quem sente que ainda não conseguiu se desprender de alguém, de uma fase ou de uma versão antiga de si mesmo.

## 🌅 Um convite

Você não precisa esquecer para seguir em frente.  
Você só precisa se reencontrar.
